# Project Overview

A command-line tool that generates name-like anagrams from words and phrases. Given any input text, it rearranges **every letter** into fictional character names that sound plausible and pronounceable.

## Commands

- Lint: `ruff check . --fix`
- Format: `ruff format .`
- Test: `pytest`
